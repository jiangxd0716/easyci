系统环境：Centos7

一、下载jdk8.0

jdk官方网站：https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

 

下载适合版本的jdk。我的linux是64位，这里我选择的是jdk-8u221-linux-x64.tar.gz。

点击登录下载或者从下方百度云链接下载。

jdk8u131： wget --no-check-certificate --no-cookies --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/jdk-8u131-linux-x64.tar.gz

百度云： 链接：https://pan.baidu.com/s/1vlCbdVmjKi9gL0Hb6O28Lg    提取码：rnad 

二、解压tar包

1.将下载的tar包通过ftp上传到服务器，这里我放在/home路径下

1 [root@localhost home]# ls
2 jdk-8u221-linux-x64.tar.gz

2.解压

1 [root@localhost home]# tar -zxvf jdk-8u221-linux-x64.tar.gz 
　　……
2 [root@localhost home]# ls
3 jdk1.8.0_221  jdk-8u221-linux-x64.tar.gz

将jdk1.8.0_221重命名为jdk

1 [root@localhost home]# mv jdk1.8.0_221/ jdk
2 [root@localhost home]# ls -al
3 总用量 190524
4 drwxr-xr-x.  3 root root        51 9月  18 13:08 .
5 dr-xr-xr-x. 17 root root       224 9月  16 14:02 ..
6 drwxr-xr-x.  7   10  143       245 7月   4 19:37 jdk
7 -rw-r--r--.  1 root root 195094741 9月  18 12:57 jdk-8u221-linux-x64.tar.gz

三、设置jdk的环境变量

  • vi /etc/profile

按shift+g跳转到文档的末尾，按o进行编辑添加jdk的环境变量

1 export JAVA_HOME=/home/jdk     //此处为jdk的文件路径
2 export JRE_HOME=${JAVA_HOME}/jre
3 export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
4 export PATH=${JAVA_HOME}/bin:$PATH

重新初始化profile配置文件

  •  . /etc/profile
1 [root@localhost home]# . /etc/profile
2 [root@localhost home]# java -version
3 java version "1.8.0_221"
4 Java(TM) SE Runtime Environment (build 1.8.0_221-b11)
5 Java HotSpot(TM) 64-Bit Server VM (build 25.221-b11, mixed mode)

jdk8.0安装成功！
